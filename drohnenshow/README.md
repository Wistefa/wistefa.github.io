# 🚁 Drohnenshow Mockup Tool

Visuelles Tool zum Planen und Simulieren von Drohnenshow-Formationen mit Übergängen.

**Live:** https://wistefa.github.io/drohnenshow  
**Teil von:** https://wistefa.github.io

## Features

- Drohnen-Positionen im 2D/3D-Viewer platzieren
- Formationen definieren und benennen
- Übergänge zwischen Formationen animieren
- Vorschau der Show-Sequenz

## Technik

- Reines HTML/JS, kein Build-Prozess, kein Service Worker
- Canvas-basierte Visualisierung

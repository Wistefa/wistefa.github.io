# 💪 Fitness-App – Tägliches Programm

Geführtes tägliches Fitnessprogramm in fünf Bereichen für Körper und Geist — jeden Morgen in ca. 40 Minuten.

**Live:** https://wistefa.github.io/fitness-app  
**Teil von:** https://wistefa.github.io

## Features

- 5 Übungsbereiche (Kraft, Ausdauer, Beweglichkeit, Balance, Entspannung)
- Geführte Übungssequenz mit Timer
- Täglicher Trainingsplan
- Fortschritts-Tracking

- Reines HTML/CSS/JS, kein Build-Prozess, kein Service Worker

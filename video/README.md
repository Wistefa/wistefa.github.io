# 🎬 SmartCut – Video Mockup

Mockup-Tool für Video-Schnitt und Export-Workflows.

**Live:** https://wistefa.github.io/video  
**Teil von:** https://wistefa.github.io

## Features

- Mediathek-Übersicht mit Thumbnails
- Timeline-Editor (Tracks, Clips)
- Export-Simulation mit Fortschrittsanzeige

## Hinweis

Dies ist ein UI-Mockup / Prototyp ohne echte Videoverarbeitung im Browser.

## Technik

- Reines HTML/CSS/JS, kein Build-Prozess, kein Service Worker

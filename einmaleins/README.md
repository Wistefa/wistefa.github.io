# ✖️ Einmaleins – Mathe lernen macht Spaß!

Interaktive Einmaleins-Lern-App für Kinder.

**Live:** https://wistefa.github.io/einmaleins  
**Teil von:** https://wistefa.github.io

## Features

- Einmaleins-Aufgaben (1×1 bis 10×10)
- Sofortiges Feedback bei richtiger/falscher Antwort
- Punktestand und Fortschritt
- Spielerische Gestaltung für Kinder

## Technik

- Reines HTML/CSS/JS, kein Build-Prozess, kein Service Worker
- Keine Datenpersistenz (Session-basiert)

# 🧰 Wilfried's Werkzeugkiste – App-Galerie

Startseite und Übersicht aller persönlichen Web-Apps auf wistefa.github.io.

**Live:** https://wistefa.github.io  
**GitHub:** https://github.com/Wistefa/wistefa.github.io

## Inhalt

Kategorisierte Kachelansicht aller verfügbaren Apps:
- Finanzen & Haushalt (Haushaltsbuch, Spesenabrechnung, Vermögen, Versicherungen)
- Freizeit & Hobby (Münzsammlung, Reiseplaner, Skat Strichliste)
- Arbeit & Organisation (FreelancerCRM, WS Dashboard)
- Spiele & Sport (KVU Storchencup)
- Tools & Sonstiges (Drohnenshow, Einmaleins, Fitness-App, Rezepte, SmartCut)

## Technik

- Reines HTML/CSS/JS, kein Build-Prozess
- Statisches Deployment via GitHub Pages
